# Scanning Web Apps

Use the proxy to capture traffic, then run active scans on discovered URLs.

## Recommended Flow

1. Start the proxy.
2. Crawl the target.
3. Run an active scan on authenticated sessions.
4. Review findings in the desktop UI.

## Good Practices

- Scope your target carefully.
- Avoid destructive payloads outside a lab.
- Re-test after remediation.
